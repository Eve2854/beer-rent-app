Politica de Privacidad

Ultima actualizacion: Marzo 2026

MOND Cerveza Artesanal utiliza la informacion brindada por los usuarios unicamente para gestionar reservas y coordinar servicios.

Datos que se recopilan

Nombre, telefono, direccion, fecha y detalles del servicio. Estos datos se usan para la comunicacion y la organizacion del evento.

Uso de la informacion

Los datos se usan para confirmar pedidos, coordinar entregas y registrar eventos en calendario cuando el cliente autoriza.

Comparticion

No compartimos datos con terceros, salvo proveedores necesarios para el servicio (por ejemplo, Google Calendar si el dueño autoriza la integracion).
